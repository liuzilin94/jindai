# 数字锦带微信小程序 - 项目规范

## 1. 概念与愿景

数字锦带是一款融合藏域非遗美学与现代简约设计的微信小程序商城，核心围绕锦带定制、成品商城、文化科普三大功能。整体风格呈现千年藏域文化的神秘与精致，让用户在现代数字生活中感受传统非遗之美。交互简洁流畅，操作直觉化，让非遗走进每个人的日常生活。

## 2. 设计语言

### 2.1 美学方向
- **风格定位**：藏域非遗 + 现代简约
- **视觉隐喻**：传统藏式织锦纹理与现代扁平设计的融合，边框和装饰融入简约锦带纹样元素

### 2.2 色彩系统
```
主色调：
- 藏红 #9E2B25 （强调色，用于按钮、重要操作）
- 藏蓝 #1D2B53 （辅助色，用于标题、导航）
- 米白 #F5F0E1 （背景色，温暖舒适的阅读体验）

辅助色：
- 金色 #C9A962 （点缀，用于特殊元素、价格）
- 深灰 #333333 （正文文字）
- 浅灰 #999999 （次要文字）
- 边框灰 #E5E5E5 （分割线、边框）
- 成功绿 #52C41A
- 警告橙 #FAAD14
```

### 2.3 字体规范
- **主字体**：-apple-system, "PingFang SC", "Helvetica Neue", sans-serif
- **标题字号**：36rpx（页面标题）、32rpx（模块标题）、28rpx（卡片标题）
- **正文字号**：26rpx
- **辅助字号**：22rpx、20rpx

### 2.4 间距系统
- **基础单位**：10rpx
- **页面边距**：24rpx
- **模块间距**：30rpx
- **卡片内边距**：20rpx
- **按钮内边距**：24rpx 36rpx

### 2.5 动效规范
- **过渡时长**：300ms
- **缓动函数**：ease-out
- **点击反馈**：opacity 0.8 + scale(0.98)
- **页面切换**：slide-up 300ms
- **弹窗动画**：fade-in + scale 300ms

## 3. 页面架构

### 3.1 五大核心页面
1. **首页** (`pages/index/index`) - 宣传科普 + 功能入口 + 热门推荐
2. **定制中心** (`pages/custom/index`) - 九宫格纹样编辑 + 手绘 + 预览下单
3. **商城** (`pages/shop/index`) - 成品锦带分类展示 + 商品详情
4. **文化科普** (`pages/culture/index`) - 非遗知识 + 纹样寓意 + 编织技艺
5. **个人中心** (`pages/profile/index`) - 订单 + 定制记录 + 收藏 + 地址

### 3.2 底部导航
固定显示5个标签：首页、定制、商城、科普、我的

## 4. 功能详情

### 4.1 首页功能
- **顶部搜索框**：支持搜索成品/定制/科普内容
- **轮播图**：3-4张非遗宣传图，自动轮播
- **功能入口**：4个大图标（锦带定制、成品商城、文化科普、我的）
- **科普轻展示**：1-2个非遗小知识卡片，点击跳转详情
- **热门推荐**：3-4款热销成品锦带卡片

### 4.2 定制中心功能（核心）
- **锦带预览区**：实时展示定制效果，平面预览模式
- **九宫格编辑**：
  - 9个等大正方形格子
  - 点击格子弹出纹样选择弹窗
  - 支持清空、重选格子
- **纹样选择区**：
  - 分类：吉祥类、自然类、文化类
  - 固定纹样：孔雀纹、彩虹纹、达玛花纹、格萨尔王纹等
- **手绘功能**：
  - 弹出手绘画布
  - 画笔工具（粗细可调）
  - 颜色选择：藏红、藏蓝、黑、金
  - 橡皮擦、清空、保存
- **规格选择**：
  - 尺寸：常规款、细款、宽款
  - 材质：棉麻、尼龙、抗菌纤维
  - 配件：龙虾扣、磁吸扣、无配件
- **刻字功能**：限2-4个字，可选字体
- **价格计算**：实时计价，明码标价
- **操作按钮**：保存方案、加入购物车、立即购买、重置

### 4.3 商城功能
- **分类导航**：4大系列（日常实用、文创伴手、高端收藏、定制成品）
- **商品展示**：瀑布流布局，卡片含图片、名称、标签、价格、销量
- **筛选排序**：价格、销量、新品、纹样分类
- **商品详情**：
  - 多图展示
  - 商品信息（名称、寓意、材质、尺寸、价格）
  - 规格选择、数量选择
  - 加购、立即购买

### 4.4 文化科普功能
- **分类导航**：纹样文化、编织技艺、非遗故事、达玛女神IP
- **内容展示**：图文卡片形式
- **互动功能**：收藏、分享
- **知识问答**：选择题互动

### 4.5 个人中心功能
- **我的订单**：全部/待付款/待发货/待收货/已完成
- **我的定制**：保存的方案、历史记录
- **我的收藏**：商品/科普卡片
- **收货地址**：增删改查
- **关于我们**：非遗工坊介绍
- **设置**：意见反馈

## 6. 技术方案

### 6.1 开发框架
- **框架**：uni-app（Vue3 + TypeScript）
- **UI 适配**：基于 uni-app 的 rpx 单位实现响应式
- **状态管理**：Pinia

### 6.2 项目结构
```
src/
├── pages/                 # 页面
│   ├── index/            # 首页
│   ├── custom/           # 定制中心
│   ├── shop/             # 商城
│   ├── culture/          # 文化科普
│   ├── profile/          # 个人中心
│   └── components/       # 页面私有组件
├── components/           # 公共组件
├── static/              # 静态资源
├── styles/              # 全局样式
│   ├── variables.scss   # 样式变量
│   └── common.scss      # 公共样式
├── utils/               # 工具函数
└── store/               # 状态管理
```

### 6.3 关键实现
- 九宫格编辑：CSS Grid 布局 + Canvas 绘制
- 手绘功能：Canvas API + 触摸事件
- 实时预览：响应式数据绑定
- 价格计算：计算属性自动计算
