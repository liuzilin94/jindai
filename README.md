# 数字锦带微信小程序

一款融合藏域非遗美学与现代简约设计的微信小程序商城，核心围绕锦带定制、成品商城、文化科普三大功能。

## 功能特性

### 1. 首页
- 轮播图展示非遗宣传内容
- 功能入口快速导航
- 非遗小知识卡片
- 热门商品推荐

### 2. 定制中心（核心功能）
- 九宫格纹样编辑
- 平台固定纹样选择（吉祥类、自然类、文化类）
- 手绘纹样功能
- 规格选择（尺寸、材质、配件）
- 刻字功能
- 实时价格计算
- 保存方案/加入购物车/立即购买

### 3. 商城页面
- 商品分类导航（日常实用、文创伴手、高端收藏、定制成品）
- 瀑布流商品展示
- 价格/销量筛选排序
- 商品详情页（含纹样寓意、关联推荐）

### 4. 文化科普
- 分类导航（纹样文化、编织技艺、非遗故事、达玛女神IP）
- 图文卡片展示
- 收藏/分享功能
- 非遗知识问答互动

### 5. 个人中心
- 订单管理
- 我的定制
- 收藏管理
- 收货地址
- 关于我们/意见反馈

## 设计规范

### 色彩系统
- 藏红 #9E2B25（主色调）
- 藏蓝 #1D2B53（辅助色）
- 米白 #F5F0E1（背景色）
- 金色 #C9A962（点缀色）

### 字体
- 主字体：-apple-system, "PingFang SC", "Helvetica Neue", sans-serif

## 技术栈

- uni-app（Vue3 + TypeScript）
- 响应式布局（rpx单位）
- Canvas API（手绘功能）

## 运行项目

```bash
# 安装依赖
npm install

# 运行微信小程序
npm run dev:mp-weixin

# 运行 H5
npm run dev:h5
```

## 项目结构

```
src/
├── pages/                 # 页面目录
│   ├── index/            # 首页
│   ├── custom/           # 定制中心
│   ├── shop/             # 商城
│   ├── culture/          # 文化科普
│   ├── profile/          # 个人中心
│   ├── product-detail/   # 商品详情
│   └── order-confirm/    # 订单确认
├── components/           # 公共组件
├── static/              # 静态资源
│   ├── icons/           # 图标
│   └── styles/          # 样式
├── data/                # Mock 数据
└── utils/               # 工具函数
```

## 注意事项

1. 图标文件需要替换为实际的图片资源
2. 图片资源需要替换为实际的产品图和banner图
3. 部分页面跳转需要创建对应的目标页面
4. 手绘功能使用Canvas API，需要适配实际绘图逻辑
